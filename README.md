## 🚀 Featured Projects

### 🏅 Olympic Medal Predictor
> *Python · Pandas · NumPy · Scikit-learn · Matplotlib · Seaborn · Flask*

Trained and compared **Ridge Regression** and **Random Forest** models with GridSearchCV hyperparameter tuning on 2,000+ rows of Olympic data — hitting **R²=0.921** and **MAE=3.28**. Deployed as a Flask web app with a responsive frontend.

---

### ✈️ AI Travel Planner
> *Python · FastAPI · LangChain · FAISS · GPT-4o*

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Railway-0B0D0E?style=flat-square&logo=railway&logoColor=white)](https://travel-plann.up.railway.app/)

A LangChain GPT-4o agent that orchestrates **5 function-calling tools** to generate 2-day itineraries. Features a multi-hop RAG search chain with 3 sequential Tavily queries, a dual-memory system (per-session conversation history + FAISS vector store), and a vanilla-JS multi-session chat frontend deployed on Railway.

---

### 🧠 MIIP
> *Python · OpenAI · LangChain · ChromaDB · FAISS*

A high-precision RAG pipeline using a **two-stage retrieve-then-rerank** approach: top-20 candidates narrowed to top-3 via a HuggingFace cross-encoder. Backed by a FastAPI server with page-level source citations to reduce hallucination.

---

### 🍜 FoodTok
> *Java · Android SDK · Supabase · Retrofit · ExoPlayer · Gemini AI*

TikTok-style Android recipe discovery app with infinite-scroll video feed, a custom **max-heap recommendation engine**, and a **Trie-based ingredient autocomplete**. Integrated Google Gemini AI for per-recipe chatbots and on-demand recipe enrichment.

🥉 **3rd place** in Android Development exhibition (voted by visitors, 20+ teams)
